## PhyOS 5.x Release Notes

PhyOS is Phygrid's edge operating system designed for running applications and AI workloads at the edge. This page documents all releases, improvements, and fixes for the PhyOS 5.x series, listed in descending order (newest first).

---

**Status:** Privately Released

- **VMware OVA:** [phyos.5.2.0.amd64.ova](https://os.phygrid.com/phyos.5.2.0.amd64.ova)
- **Hyper-V image:** [phyos.5.2.0.amd64.vhdx.tar.gz](https://os.phygrid.com/phyos.5.2.0.amd64.vhdx.tar.gz)
- **QCOW2 image:** [phyos.5.2.0.amd64.qcow2.tar.gz](https://os.phygrid.com/phyos.5.2.0.amd64.qcow2.tar.gz)

- **Jetson Orin Nano Support:** Added full support for Jetson Orin Nano developer kit

## Upgrade Instructions

To upgrade to the latest PhyOS version:

```bash
# Via Phygrid CLI
phy device update <device-name>

# Or via Console
# Navigate to Devices → Select Device → System → Update OS
```

**Note:** Always review the release notes and backup your configuration before upgrading production devices.
