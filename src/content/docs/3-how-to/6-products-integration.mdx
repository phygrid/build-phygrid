## üõçÔ∏è What Are Products?

In Phygrid, Products represent the core entities for showcasing and managing inventory. Each product serves as a detailed,
structured data object that includes attributes like multilingual names and descriptions,
product types, pricing, availability across physical locations (spaces),
and configuration variants such as color or size.
Products are centrally maintained through the Phygrid Console and are designed to be integrated
seamlessly into Screen Apps and Edge Apps, enabling rich, dynamic, and data-driven customer experiences.

**Each product may include:**

- **Localization:** Support for multiple languages in names, descriptions, and statuses.

- **Space-specific status:** Products can be active or inactive across different physical spaces.

- **Variants:** Differentiations like size, color, or style under the same product group.

- **Relationships:** Links to recommended or related product groups.

- **Metadata:** Information such as sort order, availability, introduction or discontinuation dates, and shelf life.

- **Presentation logic:** Including sort-friendly names and prices to control how products appear in UI layers.

## üîß Common Use Cases and Real-Life Scenarios

**Products integration is essential for various business scenarios:**

- **Retail Displays**: Showcase products with real-time pricing and availability on digital signage.

- **Interactive Kiosks**: Allow customers to browse products, view details, and check inventory.

- **Inventory Management**: Track stock levels and update product information across multiple locations.

- **Dynamic Pricing**: Display promotional pricing, discounts, and special offers.

- **Product Catalogs**: Create comprehensive product listings with images, descriptions, and specifications.

- **Multi-location Retail**: Manage products across different stores with location-specific pricing and inventory.

## üöÄ Available Ways to Create Products

There are several methods to integrate products into your Phygrid ecosystem, each suited for different use cases and technical requirements.

### API-Based Integration

The most flexible approach for custom integrations and automated workflows.

**Best for:**

- Custom applications with specific requirements

- Automated product updates from external systems

- Real-time synchronization with existing inventory systems

**Key benefits:**

- Full control over product data structure

- Real-time updates and synchronization

- Integration with existing business systems

### Google Product Feed (GPF)

A streamlined approach for retailers using Google's product feed format.

**Best for:**

- Retailers already using Google Shopping

- Standardized product data management

- Quick setup with existing Google feeds

**Key benefits:**

- Leverages existing Google Shopping feeds

- Automatic synchronization with Google's product format

**[How to integrate?](https://learn.phygrid.com/products/integrate-products-using-google-product-feed/)**

### Console UI

The most user-friendly approach for manual product management.

**Best for:**

- Small to medium product catalogs

- Manual product entry and management

- Teams without technical expertise

**Key benefits:**

- No technical knowledge required

- Visual interface for product management

**[How to integrate?](https://learn.phygrid.com/products/create-a-product-manually/)**

### Integrate Product Information

Now let's create a React component that displays products using the SDK:

```typescript
import React, { useEffect, useState } from "react"
import { HubClient } from "@phygrid/hub-client"

interface Product {
  id: string
  title: string
  description: string
  price: number
  images: string[]
  category: string
  tags: string[]
}

const ProductDisplay: React.FC = () => {
  const [products, setProducts] = useState<Product[]>([])
  const [loading, setLoading] = useState(true)
  const [settings, setSettings] = useState<any>({})

  useEffect(() => {
    const initializeApp = async () => {
      try {
        const hubClient = new HubClient()

        // Get app settings
        const appSettings = await hubClient.settings.get()
        setSettings(appSettings)

        // Get products
        const productList = await hubClient.products.list()
        setProducts(productList)
      } catch (error) {
        console.error("Failed to initialize app:", error)
      } finally {
        setLoading(false)
      }
    }

    initializeApp()
  }, [])

  if (loading) {
    return <div>Loading products...</div>
  }

  return (
    <div className={`product-grid ${settings.displayMode}`}>
      {products.map(product => (
        <div key={product.id} className="product-card">
          {product.images.length > 0 && (
            <img src={product.images[0]} alt={product.title} />
          )}
          <h3>{product.title}</h3>
          <p>{product.description}</p>
          {settings.showPricing && (
            <div className="price">${product.price.toFixed(2)}</div>
          )}
          <div className="category">{product.category}</div>
        </div>
      ))}
    </div>
  )
}

export default ProductDisplay
```

## üîå API-Based Integration

For more advanced use cases, you can use the Products API directly to create, update, and manage products programmatically.

### Example Flow for Integration End-to-End

Here's a complete example of how to integrate products using the API:

```typescript
import { HubClient } from "@phygrid/hub-client"

class ProductManager {
  private hubClient: HubClient

  constructor() {
    this.hubClient = new HubClient()
  }

  // Create a new product
  async createProduct(productData: {
    title: string
    description: string
    price: number
    category: string
    images?: string[]
  }) {
    try {
      const product = await this.hubClient.products.create({
        ...productData,
        environment: "production",
        language: "en",
        space: "default",
      })

      console.log("Product created:", product.id)
      return product
    } catch (error) {
      console.error("Failed to create product:", error)
      throw error
    }
  }

  // Update existing product
  async updateProduct(productId: string, updates: Partial<Product>) {
    try {
      const updatedProduct = await this.hubClient.products.update(
        productId,
        updates
      )
      console.log("Product updated:", productId)
      return updatedProduct
    } catch (error) {
      console.error("Failed to update product:", error)
      throw error
    }
  }

  // Get products with filtering
  async getProducts(filters?: {
    category?: string
    tags?: string[]
    priceRange?: { min: number; max: number }
  }) {
    try {
      let products = await this.hubClient.products.list()

      // Apply filters
      if (filters?.category) {
        products = products.filter(p => p.category === filters.category)
      }

      if (filters?.tags) {
        products = products.filter(p =>
          filters.tags!.some(tag => p.tags.includes(tag))
        )
      }

      if (filters?.priceRange) {
        products = products.filter(
          p =>
            p.price >= filters.priceRange!.min &&
            p.price <= filters.priceRange!.max
        )
      }

      return products
    } catch (error) {
      console.error("Failed to get products:", error)
      throw error
    }
  }

  // Delete a product
  async deleteProduct(productId: string) {
    try {
      await this.hubClient.products.delete(productId)
      console.log("Product deleted:", productId)
    } catch (error) {
      console.error("Failed to delete product:", error)
      throw error
    }
  }
}

// Usage example
const productManager = new ProductManager()

// Create a product
const newProduct = await productManager.createProduct({
  title: "Wireless Headphones",
  description: "High-quality wireless headphones with noise cancellation",
  price: 199.99,
  category: "Electronics",
  images: ["https://example.com/headphones.jpg"],
})

// Get all electronics products
const electronics = await productManager.getProducts({
  category: "Electronics",
})
```

## üìä GPF-Based Integration

Google Product Feed integration provides a streamlined way to import and manage products using Google's standardized format.

### Example Flow for Integration End-to-End

1. **Set up GPF Integration in Console**:

   - Navigate to **Content ‚Üí Products**
   - In the **Integrations** tab, click **New Integration**
   - Select **Google Product Feed**
   - Name your integration (e.g., "Main Product Feed")
   - Set the sync schedule (e.g., every 6 hours)
   - Select your environment
   - Provide the feed URL for each language
   - Save the integration

2. **Configure Feed URL**:
   Your Google Product Feed URL should follow this format:

   ```
   https://your-domain.com/feed.xml
   ```

3. **Monitor Integration**:

   - Check the integration status in the Console
   - Review sync logs for any errors
   - Verify products are appearing correctly

4. **Use Products in Your App**:

   ```typescript
   import { HubClient } from "@phygrid/hub-client"

   const hubClient = new HubClient()

   // Products from GPF will be available through the standard API
   const products = await hubClient.products.list()

   // Filter by source if needed
   const gpfProducts = products.filter(p => p.source === "gpf")
   ```

### Different Scenarios and Use Cases

#### 1. Product Catalog Display

```typescript
// Display all products in a grid
const ProductCatalog = () => {
  const [products, setProducts] = useState([])

  useEffect(() => {
    const loadProducts = async () => {
      const hubClient = new HubClient()
      const allProducts = await hubClient.products.list()
      setProducts(allProducts)
    }
    loadProducts()
  }, [])

  return (
    <div className="catalog">
      {products.map(product => (
        <ProductCard key={product.id} product={product} />
      ))}
    </div>
  )
}
```

#### 2. Category-Based Filtering

```typescript
// Filter products by category
const CategoryFilter = ({ selectedCategory }: { selectedCategory: string }) => {
  const [products, setProducts] = useState([])

  useEffect(() => {
    const loadProducts = async () => {
      const hubClient = new HubClient()
      const allProducts = await hubClient.products.list()
      const filtered = allProducts.filter(p => p.category === selectedCategory)
      setProducts(filtered)
    }
    loadProducts()
  }, [selectedCategory])

  return <ProductGrid products={products} />
}
```

#### 3. Search Functionality

```typescript
// Search products by title or description
const ProductSearch = ({ query }: { query: string }) => {
  const [results, setResults] = useState([])

  useEffect(() => {
    const searchProducts = async () => {
      const hubClient = new HubClient()
      const allProducts = await hubClient.products.list()
      const searchResults = allProducts.filter(
        product =>
          product.title.toLowerCase().includes(query.toLowerCase()) ||
          product.description.toLowerCase().includes(query.toLowerCase())
      )
      setResults(searchResults)
    }

    if (query.length > 2) {
      searchProducts()
    }
  }, [query])

  return <ProductList products={results} />
}
```

#### 4. Price Range Filtering

```typescript
// Filter products by price range
const PriceFilter = ({
  minPrice,
  maxPrice,
}: {
  minPrice: number
  maxPrice: number
}) => {
  const [products, setProducts] = useState([])

  useEffect(() => {
    const filterByPrice = async () => {
      const hubClient = new HubClient()
      const allProducts = await hubClient.products.list()
      const filtered = allProducts.filter(
        product => product.price >= minPrice && product.price <= maxPrice
      )
      setProducts(filtered)
    }
    filterByPrice()
  }, [minPrice, maxPrice])

  return <ProductGrid products={products} />
}
```

#### 5. Real-time Product Updates

```typescript
// Listen for product changes
const ProductMonitor = () => {
  const [products, setProducts] = useState([])

  useEffect(() => {
    const hubClient = new HubClient()

    // Initial load

    loadProducts()

    // Set up real-time updates (if supported)

    return () => clearInterval(interval)
  }, [])

}
```
